== Description ==

ViperProof enables you to show various aspects of social proof on your blog, in order to help you get more feed subscribers. You can show off the number of posts you've written, how many comments your site has, how many Twitter followers you have, and how many Facebook fans you have.

There's also an option to include the number of monthly visitors you receive. If you have any questions, please check our [Wordpress Plugins](http://www.viperchill.com/wordpress-plugins/) page.

== Installation ==

Follow the steps below to install the plugin.

1. Upload the `ViperProof` folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to ViperProof settings to edit your settings.
