== Description ==

Bloggers use this to easily show other forms of 'social proof' very easily on their blog to help other people see that their site is worth reading. Using a
simple shortcode, or the widget, users can display: - How many comments their blog has - How many posts they've written - Number of Facebook fans- Number of
Twitter followers (taken automatically from account page) - Alexa rank - and Google pagerank.

== Installation ==

1. Upload ViperProof directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the options page to make changes.
